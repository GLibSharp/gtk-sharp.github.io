GtkRevealer

The GtkRevealer widget is a container which animates the transition of its child from invisible to visible.

The style of transition can be controlled with gtk_revealer_set_transition_type.

These animations respect the GtkSettings:gtk-enable-animations setting.

CSS nodes

GtkRevealer has a single CSS node with name revealer. When styling GtkRevealer using CSS, remember that it only hides its contents, not itself. That means applied margin, padding and borders will be visible even when the GtkRevealer:reveal-child property is set to FALSE.

The GtkRevealer widget was added in GTK+ 3.10.
