GtkFontChooser

GtkFontChooser is an interface that can be implemented by widgets displaying the list of fonts. In GTK+, the main objects that implement this interface are GtkFontChooserWidget, GtkFontChooserDialog and GtkFontButton. The GtkFontChooser interface has been introducted in GTK+ 3.2.

Methods

Gtk.IFontChooser.SetFilterFunc 

Gtk.IFontChooser.SetFilterFunc (Gtk.FontFilterFunc,
                                IntPtr,
                                GLib.DestroyNotify)

Adds a filter function that decides which fonts to display in the font chooser.

Parameters:

filter –

a Gtk.FontFilterFunc, or null

user_data –

data to pass to filter

destroy –

function to call to free data when it is no longer needed

Since : 3.2

Gtk.FontFilterFunc 

bool
(*Gtk.FontFilterFunc) (Pango.FontFamily,
                       Pango.FontFace,

The type of function that is used for deciding what fonts get shown in a Gtk.IFontChooser. See Gtk.IFontChooser.SetFilterFunc.

Parameters:

family –

a Pango.FontFamily

face –

a Pango.FontFace belonging to family

data –

user data passed to Gtk.IFontChooser.SetFilterFunc

Returns –

true if the font should be displayed
