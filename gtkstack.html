Gtk.Stack.set_homogeneous 

def Gtk.Stack.set_homogeneous (self, homogeneous):
    #python wrapper for 'gtk_stack_set_homogeneous'

Sets the Gtk.Stack to be homogeneous or not. If it is homogeneous, the Gtk.Stack will request the same size for all its children. If it isn't, the stack may change size when a different child becomes visible.

Since 3.16, homogeneity can be controlled separately for horizontal and vertical size, with the hhomogeneous and vhomogeneous.

Parameters:

stack ( Gtk.Stack ) –

a Gtk.Stack

homogeneous ( bool ) –

True to make stack homogeneous
