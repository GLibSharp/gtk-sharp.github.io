Gtk.CellLayout.pack_start 

def Gtk.CellLayout.pack_start (self, cell, expand):
    #python wrapper for 'gtk_cell_layout_pack_start'

Packs the cell into the beginning of cell_layout. If expand is False, then the cell is allocated no more space than it needs. Any unused space is divided evenly between cells for which expand is True.

Note that reusing the same cell renderer is not supported.

Parameters:

cell_layout ( Gtk.CellLayout ) –

a Gtk.CellLayout

cell ( Gtk.CellRenderer ) –

a Gtk.CellRenderer

expand ( bool ) –

True if cell is to be given extra space allocated to cell_layout

Since : 2.4

Gtk.CellLayout.add_attribute 

def Gtk.CellLayout.add_attribute (self, cell, attribute, column):
    #python wrapper for 'gtk_cell_layout_add_attribute'

Adds an attribute mapping to the list in cell_layout.

The column is the column of the model to get a value from, and the attribute is the parameter on cell to be set from the value. So for example if column 2 of the model contains strings, you could have the “text” attribute of a Gtk.CellRendererText get its values from column 2.

Parameters:

cell_layout ( Gtk.CellLayout ) –

a Gtk.CellLayout

cell ( Gtk.CellRenderer ) –

a Gtk.CellRenderer

attribute ( str ) –

an attribute on the renderer

column ( int ) –

the column position on the model to get the attribute from
