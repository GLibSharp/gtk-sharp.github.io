Gtk.Assistant.commit 

def Gtk.Assistant.commit (self):
    #python wrapper for 'gtk_assistant_commit'

Erases the visited page history so the back button is not shown on the current page, and removes the cancel button from subsequent pages.

Use this when the information provided up to the current page is hereafter deemed permanent and cannot be modified or undone. For example, showing a progress page to track a long-running, unreversible operation after the user has clicked apply on a confirmation page.

Enumerations

Gtk.AssistantPageType

An enum for determining the page role inside the Gtk.Assistant. It's used to handle buttons sensitivity and visibility.

Note that an assistant needs to end its page flow with a page of type Gtk.AssistantPageType.CONFIRM, Gtk.AssistantPageType.SUMMARY or Gtk.AssistantPageType.PROGRESS to be correct.

The Cancel button will only be shown if the page isn’t “committed”. See Gtk.Assistant.commit for details.
