GtkAdjustment

The GtkAdjustment object represents a value which has an associated lower and upper bound, together with step and page increments, and a page size. It is used within several GTK+ widgets, including GtkSpinButton, GtkViewport, and GtkRange (which is a base class for GtkScrollbar and GtkScale).

The GtkAdjustment object does not update the value itself. Instead it is left up to the owner of the GtkAdjustment to control the value.

Gtk.Adjustment.ClampPage 

Gtk.Adjustment.ClampPage (double,
                          double)

Updates the Value property to ensure that the range between lower and upper is in the current page (i.e. between Value and Value + PageSize). If the range is larger than the page size, then only the start of it will be in the current page.

A ValueChanged signal will be emitted if the value is changed.

Parameters:

lower –

the lower value

upper –

the upper value
